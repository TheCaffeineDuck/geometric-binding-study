# Geometric Oscillon Binding Study
**Created:** 2026-03-15

This is a reproducible study of geometric oscillon binding in sextic scalar field theory.
The field evolves under the potential V(φ) = ½m²φ² − (g4/24)φ⁴ + (g6/720)φ⁶, using
a spectral RK4 integrator with Kreiss-Oliger dissipation (see `engine/evolver.py`).

## Structure

## Three Study Phases

### Phase 1 — Single-Oscillon Reference Evolution
Evolve a single Gaussian oscillon under the validated parameters to establish a reference
energy E_single and confirm long-term stability. Provides the baseline against which all
multi-body binding energies are measured.

### Phase 2 — Corrected Binding Energy Validation
Place two oscillons at varying separations and compute the binding energy
E_bind = E_total − N × E_single over time. Validates that the binding energy calculation
is self-consistent and reproduces the sign and magnitude seen in prior studies.

### Phase 3 — Cross-Edge Universality Across Geometries
Test whether the parity-rule binding signal (bound vs. unbound) is universal across
geometric configurations: tetrahedron, cube, octahedron. Sweeps cross-edge count
as the independent variable to identify a universal threshold.

## Parameters (default_params.json)

| Parameter | Value | Description |
|---|---|---|
| m | 1.0 | Mass |
| g4 | 0.30 | Quartic coupling |
| g6 | 0.055 | Sextic coupling |
| phi0 | 0.5 | Gaussian amplitude |
| R | 2.5 | Gaussian width |
| N | 64 | Grid points per side |
| L | 50.0 | Box size |
| dt | 0.05 | Time step |
| dissipation_sigma | 0.01 | Kreiss-Oliger strength |
